# JSP_Board
JSP 게시판 만들기

DB는 MY-SQL사용하였으며
회원가입이 필요없는 자유게시판 입니다.

글 작성/수정/삭제가 가능하며
작성 시 입력한 비밀번호와 일치해야만 수정과 삭제가 이루어집니다.
